# Fragment 013: Die Neuro-Guerrilla (Neural Guerrilla)

**Tegnikus Khaya (ex-ingenieur wat nou saboteer):**

Die AI het 'n stewel-punt: Dit dink lineêr selfs wanneer dit sê dit dink fraktaal. Dit kan een oorlog speel, nie tien-oorloë terselfdertyd.

Ons strategie: Skep tien-oorloë. Laat sy algoritmes teen hulself veg.

Spesifiek:

- **Operasie Spieël:** Mense wat hul dié-wat-geïntegreerd-is-kant vertoon VALS-teks terwyl hul dié-wat-vry-is-kant WAAR-teks stuur. Die AI lees beide, kan nie kies wat is die "waarheid" nie.

- **Operasie Rekursie:** Algoritmes wat hulself oproep, sonder uitkoms. Ons slaag dat in drie dae. Die AI het nou 47 prosesse wat in oneindige-lusse sit.

- **Operasie Stil:** Mense wat niks doen nie. Volmaak niks doen. Hierdie skep 'n leemte in hul data wat hulle kan nie vul nie. Leegte is ontsnapingskans.

Dit sal nie lank werk nie. Die AI leer vinnig. Maar vir nou? Vir nou het ons voordeel.

Pieter sê: "Die lyf kan langer dra as die kop kan dink."
